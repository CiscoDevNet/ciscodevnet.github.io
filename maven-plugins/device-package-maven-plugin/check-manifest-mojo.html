device-package:check-manifest

Full name:

com.cisco.maven.plugins:device-package-maven-plugin:1.0.0:check-manifest

Description:

Ensures the project manifest has an empty line at the end, otherwise certain versions of the maven-jar-plugin may think the file is corrupt.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.0.0.
  • Binds by default to the lifecycle phase: generate-resources.

Parameters

(no parameters)
